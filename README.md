# Ayursmart
Ayurvedic_clinic_management_system

AyurSmart is a web-based Ayurveda Consultation and Management System developed to simplify and digitalize the daily operations of an Ayurvedic clinic. The system helps manage patient records, Prakriti (dosha) quiz results, appointments, remedies, and payment information in a centralized database.

The application allows patients to register, log in, take the Prakriti quiz, and book appointments for call or offline consultations. Based on the quiz results, the system identifies the dominant dosha (Vata, Pitta, or Kapha) and provides suitable Ayurvedic remedies and recommendations.

The system is developed using HTML, CSS, and JavaScript for the frontend, Python Flask for the backend, and MySQL for database management. By replacing manual record keeping with a digital platform, AyurSmart improves efficiency, accuracy, data security, and overall clinic management.
> Key Features
> Patient Registration and Login
> Prakriti Quiz Assessment
> Appointment Booking
> Remedy Management
> Payment Record Management
> Admin Dashboard
> Secure Database Storage
AyurSmart provides a simple, user-friendly, and efficient solution for managing Ayurvedic clinic activities and enhancing patient care.
